# Technology Stack

## Frontend
- **Framework:** React (Vite)
- **Styling:** Tailwind CSS
- **State Management:** React Context / Hooks
- **Charting:** Recharts
- **HTTP Client:** Axios

## Backend
- **Runtime:** Node.js
- **Framework:** Express.js
- **APIs:** 
    - Polymarket Gamma API (Market Data)
    - CryptoPanic API (News Aggregation)

## Infrastructure
- **Hosting:** Localhost (Development)
- **Package Manager:** npm
