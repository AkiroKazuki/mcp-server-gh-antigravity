# Polymarket Analysis App

## Overview
A real-time dashboard for analyzing Polymarket bets, integrating market data with relevant news to help users make informed decisions.

## Goal
To provide a consolidated view of market probabilities, trading volume, and related news, enabling users to identify profitable betting opportunities.

## Key Features
- **Market Feed:** Real-time list of active markets sorted by volume/liquidity (via Gamma API).
- **News Integration:** Contextual news articles linked to market tags (via CryptoPanic API).
- **Bet Analyzer:** Calculator for potential returns and risk assessment.
- **Charts:** Visual history of price movements.
